Introducción y Configuración del Entorno

Preparando el proyecto y las dependencias

5 min

Bienvenido a este tutorial completo sobre la creación de APIs REST con Node.js y Express. En esta primera lección, configuraremos nuestro entorno de desarrollo y crearemos la estructura base de nuestro proyecto.

Requisitos Previos

  • Conocimientos básicos de JavaScript
  • Familiaridad con conceptos de programación backend
  • Node.js instalado (versión 18 o superior)
  • Un editor de código (recomendamos VS Code)

Configuración Inicial

Primero, vamos a crear un nuevo directorio para nuestro proyecto:

mkdir api-rest-tutorial
cd api-rest-tutorial
npm init -y

Instalemos las dependencias necesarias:

# Dependencias principales
npm install express mongoose dotenv cors helmet
npm install bcryptjs jsonwebtoken express-validator

# Dependencias de desarrollo
npm install -D nodemon concurrently

Estructura del Proyecto

Crearemos la siguiente estructura de carpetas:

api-rest-tutorial/
├── config/
│   └── database.js
├── controllers/
├── middleware/
├── models/
├── routes/
├── utils/
├── .env
├── .gitignore
├── package.json
└── server.js
Tip: Esta estructura nos permitirá mantener nuestro código organizado y seguir las mejores prácticas de Node.js.

¿Qué aprenderemos?

  • • Configuración de Express.js
  • • Conexión a MongoDB con Mongoose
  • • Creación de modelos de datos
  • • Implementación de rutas RESTful
  • • Autenticación con JWT
  • • Middleware de seguridad
  • • Validación de datos
  • • Testing y documentación
